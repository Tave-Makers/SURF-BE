## 특정 팀 내 멤버별 점수 현황 조회

### 개요

특정 팀에 속한 멤버들의 개인별 점수 현황을 조회합니다.

### 📌 엔드포인트

`GET /v1/admin/scores/teams/{teamId}/members`

### 🔐 인증

- **인증 필요 여부:** JWT 인증 필요 (관리자 권한)

> 요청 헤더(Header)에 아래와 같이 Authorization 필드를 포함해야 합니다.
>
> `Authorization: Bearer {JWT_TOKEN}`

### ▶️ 요청 (Request)

<aside>

### Headers

| Key | Type | 설명 | 필수 |
| --- | --- | --- | --- |
| `Authorization` | String | `Bearer {JWT_TOKEN}` | ✅ |

### Path Parameters

| Key | Type | 설명 |
| --- | --- | --- |
| `teamId` | Long | 조회할 팀의 고유 ID |

### ▶️요청 예시

```
GET /v1/admin/scores/teams/1/members
```

</aside>

---

---

### ◀️ 응답 (Response)

<aside>

### 성공 (Success)

| HTTP Status | 의미 |
| --- | --- |
| `200 OK` | 팀 멤버 점수 현황 조회 성공 |

**Body**

| Key | Type | 설명 |
| --- | --- | --- |
| `code` | int | HTTP 상태 코드 |
| `message` | String | 응답 메시지 |
| `data.teamId` | Long | 팀 ID |
| `data.teamName` | String | 팀 이름 |
| `data.members` | Array | 팀 멤버별 점수 목록 |
| `data.members[].memberId` | Long | 회원 ID |
| `data.members[].profileImageUrl` | String | 프로필 이미지 URL |
| `data.members[].name` | String | 회원 이름 |
| `data.members[].part` | String | 파트명 (예: "백엔드", "프론트엔드") |
| `data.members[].rewardTotal` | BigDecimal | 상점 합계 |
| `data.members[].penaltyTotal` | BigDecimal | 벌점 합계 (양수) |
| `data.members[].totalScore` | BigDecimal | 총 점수 |

### 실패 (Error)

<aside>

| HTTP Status | 의미 | 에러 코드 |
| --- | --- | --- |
| `401 Unauthorized` | 인증 실패 | `UNAUTHORIZED` |
| `403 Forbidden` | 관리자 권한 필요 | `FORBIDDEN` |
| `404 Not Found` | 존재하지 않는 팀 | `TEAM_NOT_FOUND` |

**응답 예시 (404 Not Found)**

```json
{
  "code": 404,
  "message": "존재하지 않는 [팀]입니다."
}
```

</aside>
